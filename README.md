# VenomStrike

VenomStrike is a browser-extension project focused on web-security validation workflows for authorized security testing and internal review.

## Overview

This public repository contains public-facing documentation, security policies, changelog notes, and demo screenshots for the VenomStrike project.

## What is included

- Project overview and documentation
- Privacy, security, and support policies
- Release history and disclaimer information
- Demo screenshots for public review

## Distribution

The production extension build is distributed through the official browser store listing when available.

## Contact

For private distribution, source access, or security reporting, please use the private repository and the documented security channel.
