# Support

## Before requesting support

1. Confirm that you are using the latest authorized build.
2. Remove older VenomStrike versions from the browser.
3. Reproduce the issue on an authorized local lab when possible.
4. Check the popup, page, and service-worker consoles.
5. Remove credentials, tokens, cookies, and personal data from all attachments.

## Where to ask

- Product usage and installation: private repository Discussions or Issues
- Reproducible bugs: use the bug issue template
- Detector accuracy: use the detector-accuracy template
- Security vulnerabilities in VenomStrike: follow `SECURITY.md`

The maintainer profile is `https://github.com/jagat1111`.
